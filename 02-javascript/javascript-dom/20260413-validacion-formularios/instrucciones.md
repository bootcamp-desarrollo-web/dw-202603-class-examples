Validación de formularios
Ejercicio HTML + JavaScript
===========================
Tu tarea es crear un formulario donde el usuario tenga que introducir un número entre 10 - 20.  
El formulario cuenta con:
- Un campo para el número con un label
- Un botón para enviar el número

En JavaScript tienes que captar el evento y comprobar que:
- El valor introducido es numérico (`isNaN()`)
- El número está dentro del rango

En caso que la validación falla, hay que imprimir un mensaje a consola. En caso contrario, imprimir a la consola un mensaje con el número introducido.
