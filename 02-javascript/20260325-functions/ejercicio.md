# Práctica de funciones
Crea una función que recibe un número.
Primero hay que validar que el número es mayor a cero - en caso que no, se imprime un error y se devuelve `-1`.  

En caso que se trata de un número positivo, se devuelve la suma de todos los números positivos entre 1 y este número.

Por ejemplo:  
`sumarHasta(5)` devuelve 1+2+3+4+5
