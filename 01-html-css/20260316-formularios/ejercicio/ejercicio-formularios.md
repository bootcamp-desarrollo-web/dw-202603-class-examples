## Práctica - formularios

Tienes que desarrollar un formulario para terminar una compra.
El formulario tiene que tener los campos siguientes:

1. Nombre y apellido (2 campos de texto)
2. Dirección de envío (campo de texto)
3. Método de pago (elegir entre Tarjeta de crédito / Bizum / PayPal) (radio button)
4. Elegir método de envío (normal / express / teletransporte) (select) (opcional)
5. Campo de comentarios (textarea)
6. Estar de acuerdo con la política de envíos (checkbox)

## Pasos a seguir:
1. Escribe el HTML, teniendo en cuenta:
    - Los nombres que eliges para los diferentes campos
    - Las etiquetas (label) que corresponden a cada campo
2. Escribe un HTML de destino para recibir los datos
3. Añade CSS para que:
    - El formulario quede centrado
    - Las etiquetas (labels) ocupen la misma anchura, y que estén alineadas a la derecha
