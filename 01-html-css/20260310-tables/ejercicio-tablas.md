# Tablas HTML + CSS

Crea una página web con HTML y CSS que muestre una tabla con estas características:

- Debe incluir encabezados (`<th>`) y celdas (`<td>`).

- Usa al menos un colspan y un rowspan para combinar celdas.

- Aplica estilos básicos con CSS (bordes, colores o espaciado).

## Ejemplo:
```
+----------+-----------+-----------+-----------+-----------+-----------+
|  Hora    |   Lunes   |  Martes   | Miércoles |  Jueves   |  Viernes  |
+----------+-----------+-----------+-----------+-----------+-----------+
| 08:00    |                  Desayuno                                 |
+----------+-----------+-----------+-----------+-----------+-----------+
| 10:00    | Yoga      |           |           |           | Compras   |
|          |           +-----------+-----------+-----------+-----------+
|          |           |       Paseo por el parque         |           |
+----------+-----------+-----------+-----------+-----------+-----------+
| 12:00    |  Comida   | Trabajo   | Trabajo   | Amigos    |           |
+----------|   con     |-----------+-----------+-----------+-----------+
| 14:00    |  amigos   |            Comida en casa                     |
+----------+-----------+-----------+-----------+-----------+-----------+
```
